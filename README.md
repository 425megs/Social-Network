# Social-Network

## User Story

```md
AS A social media startup
I WANT an API for my social network that uses a NoSQL database
SO THAT my website can handle large amounts of unstructured data
```

---

## Overview
This API is mocking a social network back-end. Users are able to create new users, edit users and friends, add/remove friends and leave/delete reactions on posts.
<br><br>

## Installation 
1. Clone this repo to your local computer
2. Open the repo in VSCode and run 'npm i', please make sure you have MongoDB installed on your computer
3. In your terminal, run 'node server.js' to launch the application
<br><br>
